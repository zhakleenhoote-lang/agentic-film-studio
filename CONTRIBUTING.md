# Contributing to Agentic Film Studio

🎬 Welcome — and thank you for your interest in Agentic Film Studio!

This project is an open-source, multi-agent film production pipeline powered by AI. Whether you're fixing a bug, suggesting a feature, improving docs, or sending a pull request — contributions of all shapes and sizes are appreciated.

---

## How to Contribute

### 🐛 Bug Reports
Open a [Bug Report](.github/ISSUE_TEMPLATE/bug.md) with a clear description, steps to reproduce, and environment details.

### ✨ Feature Suggestions
Open a [Feature Request](.github/ISSUE_TEMPLATE/feature.md) describing the use case and any alternatives you've considered.

### 📖 Documentation
Found a typo, unclear section, or missing docs? Open a [Documentation Improvement](.github/ISSUE_TEMPLATE/docs.md) or send a direct PR.

### 🔧 Code / Pull Requests
See the workflow below.

---

## Pull Request Workflow

1. **Fork** the repository
2. **Create a branch**: `feat/your-feature-name` or `fix/your-bugfix-name`
3. **Commit** your changes with clear, descriptive messages
4. **Push** to your fork
5. **Open a Pull Request** using [our template](.github/PULL_REQUEST_TEMPLATE.md)

### PR Requirements

- ✅ CI pipeline passes (lint + test)
- ✅ At least one maintainer review and approval
- ✅ No merge conflicts with `main`
- ✅ All checklist items in the PR template are addressed
